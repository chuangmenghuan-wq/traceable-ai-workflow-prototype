# Small-Test Value Capability Checklist

**Status:** Reusable checklist  
**Task ID:** LEAD-FLOW-003K  
**Date:** 2026-07-07

Use this checklist when creating a new small-test value example for another industry or workflow.

---

## Required pieces

```text
[ ] Summary card exists.
[ ] Value card exists.
[ ] Before-after example exists.
[ ] The example is simple.
[ ] The first test stays small.
[ ] A person checks the result.
[ ] The value is visible without a live explanation.
```

---

## Output shape

```text
Summary card: one quick overview.
Value card: four short lines.
Before-after example: one simple before, test, after, value flow.
```

---

## Done means

```text
A reader can understand what changes after the small test.
```
