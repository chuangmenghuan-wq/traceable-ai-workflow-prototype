# Safety Boundaries

This repository is a public-safe prototype.

Public-safe means the repository is designed for external review without private operational material or real-world sensitive records.

## Allowed in this repo

- synthetic tasks
- fictional scenarios
- mock reports
- sample trace logs
- prototype documentation

## Not included

- real user or organization records
- private operational notes
- production connections
- live system access
- confidential materials

## Principle

Show the workflow, not the private case.

All outputs should be treated as prototype examples that require human review before real-world use.
