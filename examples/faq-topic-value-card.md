# FAQ Topic Value Card

**Status:** Public sample card  
**Task ID:** LEAD-FLOW-003Q  
**Date:** 2026-07-07

This card explains a small first test for repeated FAQ-style questions.

---

## Value card

```text
Before: similar questions arrive in different wording.
During: AI groups repeated questions into simple topics.
After: a person checks the grouped result.
Value: the team can prepare clearer replies for the most repeated topics first.
```

## Example topics

```text
Time questions
Location questions
Change questions
Item questions
Status questions
```
