# Appointment-Service Value Card

**Status:** Public sample card  
**Task ID:** LEAD-FLOW-003L  
**Date:** 2026-07-07

This card explains a small first test for an appointment-based service.

---

## Value card

```text
Before: appointment questions arrive in many different forms.
During: AI groups similar questions into simple topics.
After: a person checks the grouped result.
Value: the team can prepare clearer replies for repeated appointment topics.
```

---

## Example topics

```text
Opening hours
Available time slots
Rescheduling
Preparation before visit
Location or parking
```
