# FAQ Topic Compact Example

**Status:** Public sample example  
**Task ID:** LEAD-FLOW-004C  
**Date:** 2026-07-07

This compact example shows a small topic-grouping result.

---

## Input idea

```text
Several similar questions appear with different wording.
```

---

## Small test

```text
Use 10 simple questions.
Group them into short topic names.
Check the result.
```

---

## Output example

```text
Time topic
Place topic
Change topic
Item topic
Progress topic
```

---

## Value

```text
The team can see repeated topics more clearly.
The team can decide which replies to prepare first.
```
